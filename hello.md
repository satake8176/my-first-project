## What is Markdown? / Markdownとは？

Markdown is a lightweight markup language used to format plain text.  
It is widely used for README files, documentation, and notes.

Markdownは、プレーンテキストを整形するための軽量マークアップ言語です。  
READMEファイル、ドキュメント、メモなど幅広く使われています。

---

## Markdown Features / Markdownの主な機能

### Text Formatting / 文字の装飾

- **Bold / 太字**
- *Italic / 斜体*
- ~~Strikethrough / 取り消し線~~

### Lists / リスト

**Unordered / 箇条書き:**
- Apple / りんご
- Banana / バナナ
- Cherry / さくらんぼ

**Ordered / 番号付き:**
1. First / 最初
2. Second / 次
3. Third / 最後

### Code / コード

Inline code: `print("Hello, World!")`

Code block:
```python
# This is a Python example / Pythonの例
name = "Yukinobu"
print(f"Hello, {name}!")
```
